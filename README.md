# GPT Colony

Первая версия RimWorld-подобной игры с базовой экономикой, нуждами и событиями.

## Запуск для ребёнка 5 лет (очень просто)

Представь, что ты открываешь коробку с игрушкой:

1. Открой папку с игрой.
2. Нажми на кнопку запуска (в терминале введи одну команду):

```bash
./run.sh
```

3. Открой в браузере адрес: http://localhost:8000
4. Готово! Игра запустилась.

## Если кнопка не сработала

Попробуй запустить так:

```bash
python3 -m http.server 8000
```

Если у тебя нет `python3`, попробуй:

```bash
python -m http.server 8000
```

Потом открой: http://localhost:8000

## Быстрый старт (через npm, опционально)

Если у вас есть доступ к npm registry:

```bash
npm install
npm run dev
```

Откройте адрес, который выведет Vite (обычно http://localhost:5173).

## Если что-то пошло не так

- **Пишет, что порт занят**: запусти так `PORT=8001 ./run.sh` и открой http://localhost:8001
- **Не находится Python**: установи Python 3 с https://python.org и повтори запуск.

## Механики первой версии

- Изометрическая карта с природными ресурсами.
- Колонисты: голод, отдых, настроение, простая автономия.
- Сбор дерева и ягод, доставка на склад.
- События: торговцы, гроза, налёт.
- Заказ постройки лагеря (клик по карте, 10 дерева).
